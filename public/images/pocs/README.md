# POC Images

Add your POC thumbnail and hero images here.

## Naming Convention

For each POC, create:
- `{slug}-thumb.png` - Card thumbnail (16:9 aspect ratio, ~800x450px recommended)
- `{slug}-hero.png` or `{slug}-hero.gif` - Hero image for detail page
